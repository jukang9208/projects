# 서울시 자치구별 초미세먼지(PM-2.5) 격차 분석

서울시 25개 자치구의 녹지·공원 현황이 초미세먼지(PM-2.5) 농도에 미치는 영향을 통계적으로 검정하고, 2019~2024년 감소 추세를 분석한 프로젝트입니다.

## 분석 배경

미세먼지는 서울 시민의 주요 환경 문제 중 하나입니다. 자치구마다 녹지와 공원 비율이 다르고, 체감상 "녹지가 많은 동네가 공기가 좋다"는 인식이 있습니다. 이 인식이 실제 데이터로 뒷받침되는지, 그리고 최근 수년간 미세먼지가 실제로 줄고 있는지를 가설검정 기반으로 검증했습니다.

## 데이터

| 데이터 | 출처 | 기간 | 단위 |
|---|---|---|---|
| 서울시 대기오염(구별) 통계 | 서울 열린데이터광장 | 2015~2024 | 월별 |
| 서울시 녹지현황 통계 | 서울 열린데이터광장 | 2015~2024 | 연간 |
| 서울시 공원(공원율) 통계 | 서울 열린데이터광장 | 2015~2024 | 연간 |

## 가설 및 검정 방법

### H1. 녹지 면적이 많은 자치구는 PM-2.5가 낮다
- 자치구를 녹지면적 중앙값 기준으로 고녹지/저녹지 두 그룹으로 분류
- Shapiro-Wilk 정규성 검정 → Levene 등분산 검정 → 독립표본 t-test

### H2. 계절에 따라 PM-2.5 평균에 유의한 차이가 있다
- 봄(3~5월), 여름(6~8월), 가을(9~11월), 겨울(12~2월) 4개 그룹
- One-way ANOVA → 유의 시 Tukey HSD 사후검정

### H3. 2019~2024년 서울시 PM-2.5가 유의하게 감소했다
- 연간 평균 PM-2.5를 연도에 대해 단순선형회귀 적용
- 기울기의 유의성(p-value)으로 감소 추세 검정

## 분석 결과

### H1 — 귀무가설 기각 불가 (비유의)
```
고녹지 평균: 20.48 ㎍/㎥ | 저녹지 평균: 19.96 ㎍/㎥
t = 1.894, p = 0.0709
```
두 그룹 간 차이는 0.52㎍/㎥에 불과하며 통계적으로 유의하지 않습니다. 녹지면적과 PM-2.5의 상관계수는 r = 0.167로, 오히려 약한 양의 방향을 보였습니다. 회귀분석 R² = 0.007로 녹지·공원율이 PM-2.5 분산을 거의 설명하지 못합니다.

**해석**: 서울시는 자치구 경계를 넘어 대기가 공유되는 도시권 환경입니다. 한 자치구의 녹지가 그 구만의 PM-2.5를 낮추기 어려운 구조이며, 교통·산업·기상 등 광역 요인이 녹지 효과를 압도합니다.

### H2 — 귀무가설 기각 (유의)
```
봄: 23.24 | 여름: 15.68 | 가을: 15.36 | 겨울: 26.42 (㎍/㎥)
F = 445.41, p ≈ 0.000
```
계절 간 PM-2.5 차이는 매우 유의합니다. Tukey HSD 사후검정에서 모든 계절 쌍이 유의한 차이를 보였습니다. F값 445는 계절이 PM-2.5의 가장 강력한 설명변수임을 나타냅니다.

**해석**: 겨울 난방 연료 사용 증가, 기온역전층에 의한 오염물질 정체, 봄철 중국발 황사가 복합적으로 작용합니다. 봄 박스플롯에서 50㎍/㎥를 넘는 이상치가 다수 관찰되는데 황사 이벤트를 반영합니다.

### H3 — 귀무가설 기각 (유의한 감소)
```
slope = -1.206 ㎍/㎥/yr, R² = 0.736, p = 0.0289
2019: 25.1 → 2020: 20.7 → 2022: 18.4 → 2024: 17.6 (㎍/㎥)
```
2019~2024년 동안 연간 약 1.2㎍/㎥씩 유의하게 감소했습니다. R² = 0.736으로 연도가 PM-2.5 변화의 73.6%를 설명합니다.

**해석**: 2019년 12월 미세먼지 계절관리제 시행, 2020~2021년 코로나19로 인한 교통·산업 활동 감소, 석탄발전 감축 정책이 복합 작용한 결과로 해석됩니다. 특히 2019 → 2020 구간(-4.4㎍/㎥)에서 급격한 감소가 나타났습니다.

## 종합 인사이트

서울시 PM-2.5는 **자치구 단위 녹지 현황보다 계절(기상)과 연도별 거시 요인(정책·산업 변화)이 훨씬 강한 결정 요인**입니다. 이는 미세먼지 저감이 개별 자치구의 녹지 확충보다 광역 정책 수준에서 접근해야 효과적임을 시사합니다.
